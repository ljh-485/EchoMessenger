## 실행 화면 (과제2)
-과제 2 코드의 실행 스크린샷 

![과제 2 실행화면](img/EM_2-1.png)
![과제 2 실행화면](img/EM_2-2.png)
![과제 2 실행화면](img/EM_2-3.png)
![과제 2 실행화면](img/EM_2-4.png)

-과제 내용
 -전송이 끝나면 입력창에 남겨진 기존 메시지를 삭제합니다.
 -전송 후에 마우스로 입력창을 다시 클릭하지 않아도 되도록 커서를 자동으로 입력창에 둡니다.
 -마우스 클릭 대신 키보드의 Enter 키를 눌러도 메시지가 전송되도록 합니다.
 -내용이 없는 빈 문자열이나 공백(Space)만 있을 때에는 메시지가 전송되지 않도록 방지합니다.
-구현 기능과 내용 설명
 -입력 후에 입력창에 마우스 커서가 자동으로 위치하여 다음 메시지를 바로 입력할 수 있다.
 -키보드의 Enter 키를 눌러도 메시지가 전송된다.
 -공백을 입력하거나 아무것도 입력하지 않고 전송하려고 하면 메시지가 전송되지 않고 입력창이 초기화된다.



## 실행 화면 (과제3)
-과제 3 코드의 실행 스크린샷

![과제 3 실행화면](img/EM_3-1.png)
![과제 3 실행화면](img/EM_3-2.png)
![과제 3 실행화면](img/EM_3-3.png)
![과제 3 실행화면](img/EM_3-4.png)

-과제 내용
 -메시지 앞에 현재시간([14:20:05])을 자동으로 결합하여 리스트에 출력합니다.
 -현재 리스트에 쌓인 총 메시지 개수를 계산하여 하단 Label에 실시간으로 업데이트합니다. 
 -사용자가 입력한 메시지의 앞뒤 불필요한 공백을 Trim() 함수로 제거하여 저장합니다.

-구현 기능과 내용 설명
 -메시지 앞에 현재 시간이 자동으로 결합되어 리스트에 출력된다.
 -총 메시지 개수가 "총 메시지 개수: N개" 형식으로 하단 Label에 실시간으로 업데이트된다.
 -공백을 포함한 메시지를 입력하면 앞뒤 불필요한 공백이 제거된 메시지가 리스트에 저장되고 출력된다.

-과제 내용 
 -ListBox에서 특정 메시지를 마우스로 클릭하고 '삭제'버튼을 누르면 해당 항목만 목록에서 제거합니다. 
  (단, 선택하지 않고 삭제 시 발생하는 에러를 예외처리 해야함)
 -'대화 기록 삭제' 버튼을 클릭하면 리스트의 모든 내용을 한번에 지웁니다.

-구현 기능과 내용 설명
 -ListBox에서 특정 메시지를 마우스로 클릭하고 '삭제'버튼을 누르면 해당 항목만 목록에서 제거된다. 
 -메시지를 선택하지 않고 삭제 버튼을 누르면 "삭제할 메시지를 먼저 선택해주세요."라는 경고 메시지가 표시된다.
 -대화 기록 삭제 버튼을 클릭하면 리스트의 모든 내용이 한번에 지워진다.
 -입력창에 50자가 넘는 메시지 작성 시, "메시지는 최대 50자까지만 입력 가능합니다."라는 경고 메시지가 표시되고 전송이 차단된다.
